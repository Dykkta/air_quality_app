Detailed Description

Klasa zarządzająca wyświetlaniem wykresów danych jakości powietrza.

Klasa ChartManager dostarcza funkcjonalność do przechowywania, przetwarzania i wizualizacji danych pomiarowych jakości powietrza. Umożliwia wyświetlanie wielu parametrów na jednym wykresie, z automatycznym doborem kolorów i osi. Zapewnia interaktywne funkcje jak podpowiedzi po najechaniu na punkty danych.
