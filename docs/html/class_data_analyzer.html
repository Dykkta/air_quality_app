Detailed Description

Klasa odpowiedzialna za analizę danych pomiarowych.

Klasa DataAnalyzer wykonuje obliczenia statystyczne na danych pomiarowych w formacie JSON. Oblicza wartości średnie, minimalne i maksymalne dla parametrów oraz określa zakres dat, w których zostały zebrane dane.
