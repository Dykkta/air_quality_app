Detailed Description

Klasa do obsługi zapytań HTTP.

Klasa HttpClient umożliwia wykonywanie podstawowych operacji sieciowych, w tym wysyłanie żądań GET oraz sprawdzanie dostępności połączenia sieciowego. Wszystkie metody są statyczne, co pozwala na ich użycie bez tworzenia instancji klasy.

Metoda wysyła synchroniczne żądanie GET do określonego URL i zwraca otrzymaną odpowiedź. Obsługuje protokoły HTTP i HTTPS. Przykład użycia:

std::string response = HttpClient::getRequest("https://api.example.com/data");

}
