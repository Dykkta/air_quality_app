\doxysubsection{Detailed Description}
Klasa do parsowania i walidacji danych JSON. 

Klasa \doxylink{class_json_handler}{Json\+Handler} zapewnia funkcjonalność do przetwarzania ciągów znaków w formacie JSON do obiektów klasy nlohmann\+::json. Wykorzystuje parser biblioteki nlohmann/json do efektywnego przetwarzania danych.
